🔐 8 Digit Password Generator

A simple and user-friendly web application that generates random 8 digit passwords instantly. The project is designed with a clean interface and fast performance to help users quickly create passwords for different accounts.

✨ Features

✔️ Generate random 8 digit passwords
✔️ Simple and clean UI
✔️ Fast password generation
✔️ Responsive design
✔️ Easy to use

🛠️ Technologies Used
🌐 HTML
🎨 CSS
⚡ JavaScript

🎯 Usage

1️⃣ Open the application
2️⃣ Click the Generate Password button
3️⃣ Instantly get a random 8 digit password

🔮 Future Improvements
📋 Copy password button
🔢 Custom password length
🔒 Strong password options
